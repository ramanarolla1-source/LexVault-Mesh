  LexVault MeshPrivate Legal Discovery & Evidence IntelligenceLexVault Mesh is a fully local, multi-agent legal discovery platform that transforms fragmented evidence into structured case knowledge using the QVAC SDK. By distributing AI workloads across consumer devices through P2P delegation, LexVault Mesh ensures absolute data sovereignty without cloud reliance.ArchitectureLexVault Mesh orchestrates intelligence across:Capture (Mobile): Local OCR and evidence ingestion.Mesh Intelligence (P2P): Workload delegation for RAG and Deep Reasoning.Synthesis (Output): Automated chronology and litigation bundle generation.Hardware SpecificationsTo ensure reproducibility, this project was validated on the following configuration:Mobile Node: iPhone 15 Pro (A17 Pro, 8GB RAM)Laptop Node: MacBook Pro M3 (16GB RAM)Desktop Node: Custom Build (AMD Ryzen 9, NVIDIA RTX 4090 24GB VRAM, 64GB RAM)Reproducibility InstructionsEnvironment Setup: Install the QVAC SDK (@qvac/sdk) as per official documentation.  Dependencies: Run pip install -r requirements.txt.Mesh Initialization: Launch src/main.py on all nodes simultaneously to establish the P2P handshake.Data Ingestion: Use the mobile app to scan evidence; the mesh will automatically route tasks based on hardware capacity.
